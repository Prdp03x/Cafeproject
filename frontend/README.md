# Frontend

React/Vite app for customer ordering, order status, login/signup, and cafe admin views.

## Source Layout

```text
src/
  api/          Axios API client
  assets/       Imported images and sounds
  components/   Feature-grouped UI components
  hooks/        Reusable React hooks
  pages/        Route-level screens
```

## Scripts

- `npm run dev` starts Vite locally.
- `npm run build` creates a production build.
- `npm run lint` runs ESLint.
- `npm run preview` previews the production build.

## Environment

Copy `.env.example` to `.env` and adjust values when needed.
